== Description ==

Remove the version of WordPress HTML content from anywhere in the site.

1. Automatically removes the version of WordPress HTML content of your site: "The contents of this comment is the responsibility of: {AUTHOR}"

== Installation ==

1. Upload the files to the `/wp-content/plugins/` (keep the original folder plugin)

2. Activate the plugin in the Plugins interface of WordPress
